### первый этап
**1)** стандартная команда для запуска приложения
```
java -jar ConfUpravlen2-1.jar -nameAnalysisPackage "org.springframework.boot" -urlAddressRepository "https://github.com/spring-projects/spring-boot.git" -workingModeWithTestRepository "test" -versionPackage "2.7.0" -nameFileWithImageGraph "spring-boot-dependencies.png" -ModePrintDependency "tree" -maxDepthAnalysisDependencies 3
```

**2)** команда для запуска приложения и демонстрации того, что неправильно указаны параметры при запуске приложения
